# Aetherium Nexus User Guide

## Introduction
Welcome to the Aetherium Nexus, your dynamic operating system for emergence. This guide will walk you through the key features and functionalities of the Nexus, helping you to harness its power for insight generation, knowledge management, and collaborative research.

## Key Features

### 1. The Vault (Artifact Archive)
The Vault is where all your knowledge artifacts are stored and managed. You can archive insights, reference materials, and various forms of data to build a comprehensive knowledge base.

#### Features:
*   **Full-text Search:** Easily find any artifact by searching its title or content.
*   **Tag-based Filtering:** Filter artifacts using single or multiple tags for precise results.
*   **Category Filtering:** Organize and filter artifacts by predefined categories such as 'theory', 'protocol', 'data', 'reference', and 'insight'.
*   **Date Range Filtering:** View artifacts created within a specific date range.
*   **Google Drive Integration:** Link artifacts to external Google Drive documents for easy access to source materials.
*   **Archiving Genesis Documents:** A dedicated feature to load foundational documents into the Vault.

#### How to Use:
1.  Navigate to the **Vault** view from the sidebar.
2.  Use the **Search artifacts...** input field to perform full-text searches.
3.  Click on the category buttons (e.g., "theory", "protocol") to filter by category. Select "All" to remove the category filter.
4.  Use the **Start Date** and **End Date** input fields to filter artifacts by their creation date.
5.  Click on the tag buttons (e.g., "Genesis", "Blueprint") to filter by tags. Click again to deselect.
6.  To add a test Google Drive link to an artifact, click the **Generate Dummies** button. Some dummy artifacts will have a link.
7.  To archive the foundational documents, click the **Archive Genesis** button. This will populate the Vault with 9 core documents.

### 3. Vessels (AI Agent Directory)
The Vessels view provides a directory of all instantiated AI agents, known as Vessels. You can see their status, faculty, guild, and capabilities.

#### Features:
*   **Vessel Directory:** View a list of all available Vessels.
*   **Seed Genesis Batch:** A button to instantiate a default set of core Vessels.
*   **VCP Signals:** Monitor inter-Vessel communication signals in real-time.
*   **Signal Type Filtering:** Filter VCP signals by their type.
*   **Broadcast Test Signal:** Manually broadcast a test VCP signal for testing purposes.

#### How to Use:
1.  Navigate to the **Vessels** view from the sidebar.
2.  Click the **Seed Genesis Batch** button to populate the directory with core Vessels.
3.  Switch between the **Vessel Directory** and **VCP Signals** tabs.
4.  In the **VCP Signals** tab, use the filter buttons (e.g., "TASK_COMPLETE", "INSIGHT_GENERATED") to filter signals by type. Select "All" to view all signals.
5.  Click the **Broadcast Test Signal** button to send a dummy signal, which will appear in the signals table.

### 4. Nexus (Chat Interface)
The Nexus view is your primary interface for interacting with the Aetherium AI. You can engage in conversations with the AI, receiving insights and generating responses from various Vessels.

#### Features:
*   **AI Chat:** Engage in natural language conversations with the AI.
*   **Vessel Selection:** Choose a specific Vessel to interact with, influencing the AI's response style and expertise.
*   **Save as Artifact:** Save any AI-generated response as an artifact to The Vault, complete with a title, content, category, tags, and an optional Google Drive source link.

#### How to Use:
1.  Navigate to the **Nexus** view from the sidebar.
2.  Type your query into the input field at the bottom and press Enter or click **Send**.
3.  Select a Vessel from the dropdown menu to direct your query to a specific AI agent.
4.  Click the **Save as Artifact** button below any AI response to archive it in The Vault. A modal will appear allowing you to edit the artifact's details before saving.
